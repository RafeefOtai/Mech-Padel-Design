# MECH Task 2 – Padel Racket & Ball Keychain

A 3D-printable keychain design featuring a padel racket and tennis ball, created in Onshape.
> Note: The dimensions were selected to demonstrate the design concept and are not intended to represent an accurately scaled or production-ready model.

## Features
- Padel racket shape with a repeated hole pattern
- Custom text engraved on the racket face
- Ball model with seam detail
- 4 mm keyring attachment holes

## Files
- `Padel_Keychain.stl` – STL file for the final model
